Khusus Hacker Pemula :

Kamu dapat menggunakan sosial media seperti Ig, Fb, Wa, Tiktok untuk mencari informasi dari target seperti hobi, foto, dll. Setelah kamu mengisi beberapa kolom di bawah ini, kamu dapat mengeklik submit pada bagian halaman paling bawah dan kamu akan mendapatkan hasil doxing kamu.

Khusus Hacker Sepuh :

Kamu dapat menggunakan tools tc di terminal termux maupun linux. Bagi pengguna Android diwajibkan untuk menginstal termux mod, bukan termux playstore. Untuk cara instalasinya adalah:

Ketik npm install truecallerjs dan enter, lalu tunggu hingga selesai. Kemudian ketik npm install -g truecallerjs dan enter, selanjutnya ketik truecallerjs login dan masukkan nomor HP kamu serta kode verifikasi yang masuk ke SMS. Kemudian silakan tracking nomor target menggunakan CMD truecallerjs -s [number]. Contohnya truecallerjs -s +6288214764360. Jika informasi target telah muncul, silakan kamu isi beberapa kolom di bawah ini (tidak semua). Untuk informasi lebih lanjut, kamu dapat menelusuri menggunakan nama lengkapnya di sosial media seperti IG, Tiktok, FB.
